Способи підключення стилів

  1. Вбудовані стилі : Вони вписані безпосередньо у HTML-документ за допомогою атрибуту style тегів HTML.
  2. Використання тега style : Можете включити CSS-стилі безпосередньо у веб-документ, використовуючи тег style в розділі head вашого HTML-документа.
  3. Зовнішні файли CSS : Створюється окремий файл з розширенням .css і підключіть його до вашого HTML-документа за допомогою тегу link.
  4. CSS-фреймворки : Можете використовувати готові CSS-фреймворки, такі як Bootstrap, Foundation або Bulma, щоб швидко встановлювати стилі для ваших веб-сайтів. Ці фреймворки зазвичай включають збалансовані та стилізовані компоненти і готові класи для стилізації вашого контенту.
